Oberheim Matrix-1000 single patch editor

Description

This software allows to open/edit/save single patches for Oberheim Matrix-1000 synthesizer.

Since users generally own a MIDI sequencer, the editor don't have MIDI thru function.

Requirements

MacOSX 10.4,10.3, should also work on 10.2.
A Oberheim Matrix-1000.
A MIDI interface.
A Mac PPC or Intel.

Operations

First you have to configure your Matrix MIDI input and output port in preferences panel. No channel number is necessary because the software don't have MIDI thru.

If your Matrix is like mine (with bugs), you have to set Send patch for ENV1 SUSTAIN and Send Patch for ENV2 TO VCA2 because when I send the parameter using the same SYSEX as for the other parameters, ENV1 SUSTAIN doesn't work and ENV2 TO VCA2 crashes the Matrix. So when the checkboxes are on, I send the whole patch for these parameters.

Then you can get a patch from the synth with Patch->Get menu or get the edit buffer.

When you open a patch from disk, default is to not transmit it to the synth, you have to do it with Patch->Send. It allows to get the current edit buffer if something interesting have to be saved first. You can change this behavior in the Preferences panel.

Global parameters are not saved with the patch, perhaps I'll do it in the future. Since Matrix-1000 don't allow single global parameter edit, you have to get/send the whole parameters.
